# Todo App

シンプルで使いやすいToDoアプリケーション

## 機能

- ✅ タスク追加
- ✅ タスク編集
- ✅ タスク削除
- ✅ タスク完了/未完了トグル
- ✅ タスクの永続化（Vercel KV）

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS + shadcn/ui
- **データベース**: Vercel KV
- **デプロイ**: Vercel

## セットアップ

1. 依存関係をインストール
```bash
npm install
```

2. 環境変数を設定
```bash
cp .env.example .env.local
```

`.env.local` ファイルにVercel KVの認証情報を設定してください。

3. 開発サーバーを起動
```bash
npm run dev
```

## デプロイ

1. Vercelアカウントでプロジェクトを作成
2. Vercel KVストレージを作成
3. 環境変数を設定
4. デプロイ

## 使い方

1. **タスク追加**: 上部の入力フィールドにタスクを入力して「追加」ボタンをクリック
2. **タスク完了**: チェックボックスをクリックして完了状態を切り替え
3. **タスク編集**: 編集アイコンをクリックしてタスクタイトルを編集
4. **タスク削除**: 削除アイコンをクリックしてタスクを削除

タスクは未完了のものが上に、完了済みのものが下に表示されます。
